# Saira Batool - Personal Portfolio

This repository hosts the source code for my personal portfolio website. It serves as a digital resume to showcase my academic background, technical expertise, projects, and professional experience as a Computer Science undergraduate.

## Project Overview

The portfolio is designed with a focus on performance, responsiveness, and modern User Interface (UI) principles. It utilizes a custom Neon-Pink theme with Glassmorphism effects to provide a distinct visual identity while maintaining readability and user experience.

## Key Features

*   **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
*   **Modern UI/UX:** Implements custom gradients, shadows, and glassmorphism.
*   **Scroll Animations:** Integrated AOS (Animate On Scroll) library for smooth transitions.
*   **Contact Form:** Functional contact form powered by EmailJS (Serverless).
*   **Dynamic Navigation:** Smooth scrolling and mobile-responsive navigation menu.

## Technologies Used

*   **HTML5:** Semantic structure and layout.
*   **Tailwind CSS:** Utility-first CSS framework for styling and responsiveness.
*   **JavaScript (ES6):** Logic for the preloader, mobile menu, and form handling.
*   **AOS Library:** For scroll-based animations.

## Local Installation

To run this project locally on your machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/11sairabatool/My_Portfolio.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd My_Portfolio
    ```

3.  **Run the project:**
    Simply open the `index.html` file in any modern web browser. For a better development experience, use the "Live Server" extension in VS Code.
